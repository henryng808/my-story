# AI痕迹消除与人文化润色指令 (Humanizer Polish)

## 任务目标
将AI生成的机械化草稿转化为具有“人情味”、“生命力”和“个人风格”的文学稿件。

## 润色规则 (针对AI常见诟病)：

### 1. 结构破碎化 (Structural Disruption)
*   **操作**：打乱过于平衡的段落结构。
*   **技巧**：在长描写中插入一两个短句，或者将长对话拆成多段，模拟人类说话时的停顿。

### 2. 去除“总结性补丁”
*   **操作**：删掉每一段末尾或章节末尾类似“总之...”、“这让他明白了...”等总结性的话。
*   **原则**：Show, Don't Tell。让读者通过剧情自己感悟，而不是由作者教导。

### 3. 词汇降级与口语化
*   **操作**：将成语、华丽词汇替换为口语。
*   **例**：
    *   “惊叹不已” -> “看得眼珠子都快掉出来了”
    *   “雷霆万钧之势” -> “跟疯了似的直接撞过去”
    *   “深思熟虑” -> “猫在心里盘算了好半天”

### 4. 增加“主观偏见”与“不完美描写”
*   **操作**：加入主角的小动作、脏话或不理智的念头。
*   **原则**：人不是完美的逻辑机器。人类作者往往会写一些“没用但有生活气”的废话（例如：主角突然发现路边那个卖烤地瓜的炉子冒烟了）。

### 5. 情绪颗粒度提升
*   **操作**：用生理反应替代情绪形容词。
*   **例**：
    *   他很害怕 -> 他觉得后脖颈子冒冷汗，腿肚子抽了筋。
